# 接口简介
基于克隆数字人技术，将输入的单人视频，创建为数字人形象。使用数字人形象搭配音频，单次可制作时长不超过12分钟的数字人说话视频。

# 计费说明
详细结算规则请查阅 [《AI中台服务专用条款》](https://www.volcengine.com/docs/6456/70590) [《AI中台计费结算规则》](https://www.volcengine.com/docs/6456/70591)
**温馨提示：**

1. 最新价格信息，以控制台实际接入为准
2. 使用方式说明：
   1. 上传视频生成视频云id
      1. 免费试用状态：并发限制为1
      2. 正式调用状态：并发限制为1，暂不计费
   2. 创建克隆数字人形象
      * 免费试用状态：免费试用状态下无法创建新的数字人形象（可使用官方提供的测试数字人 **250623\-zhibo\-linyunzhi**）
      * 正式调用状态：并发限制为1，每购买一个资源包，就可以创建一个克隆数字人
   3. 生成数字人视频
      * 免费试用状态：可以生成200秒数字人视频，并发限制为1
      * 正式调用状态：并发限制为1，按视频生成时长计费，每秒0.1元
3. 使用建议&计费说明
* 只有调用成功（正常返回生成的数字人和视频），才会进行计费。
* 接口单次调用输出一个视频/一个数字人
